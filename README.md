* ### Máquina Diferencial ### 
Charles Babbage 1822 -> "primera computadpra del mundo"  
Utilizaba engranajes para calcular y generar tablas de funciones polinómicas de forma automática , evitando los errores humanos.

* ### Máquina Diferencial II ###
Museo de Ciencias de Londres 1991  
Siguió los pasos de Charles Babbage y funcionó correctamente, demostrando que su diseño era viable y que pude haberse construido en su época.

* ### Máquina Analítica ###
Charles Babbage, 1834-1837.  
Primera máquina de cálculo automática capaz de realizar distintos tipos de operaciones, superando así a la máquina diferencial en capacidad de análisis.  
* El Molino - The Mill: La CPU o Ptocesador -> donde se hacen las operaciones.
* El Almacén - The Store: La Memoria -> donde se guardan los números y datos.
* Las Tarjetas perforadas - Punch card system: El Software -> entrada de datos e instrucciones.

* ### Telar De Jacquard ###
Joseph Marie Jacquard 1801.  
Primer sistema de almacenamiento de info binaria. Utilizaba tarjetas perforadas para automatizar diseños. Funcionaba como un sistema binario: agujero (1 o "encendido") activaba el hilo y sin agujero (0 o "apagado") no lo movía.  
1. La separación entre Hardware y Software
Hardware: El telar en sí (máquina física, engranajes, hilos)
Software: Tarjetas perforadas
Con el telar de Joseph Marie Jacquard solo se cambiaban las tarjetas perforadas (programa), permitiendo distintos resultados sin alterar la máquina.

 2. El sistema binario
Su sistema usaba lógica binaria, siendo un antecedente del funcionamiento del transistor moderno.

3. El pixel y la imagen rasterizada
El tejido puede entenderse como una imagen tipo mapa de bits: la tela es una ciadrícula donde cada cruce actúa como píxel. Las tarjetas perforadas funcionan como un archivo digital que indica que puntos activar para formar la imagen.

* ### La primera programadora ###
#### Ada Lovelace (1815-1852) ####  
Matemática y escritora británica conocida por su trabajo en el campo de programación y por su colaboración con Charles Babbage. Comprendió el verdadero potencial de la Máquina Analítica, no solo procesar números, sino manipular cualquier tipo de información (como música y arte), inventando así las bases de la programación moderna.  
* El primer algortimo
Entre 1842 y 1843, Ada Lovelace escribió algoritmos para la Máquina Analítica de Charles Babbage, considerandos el primer programa de la historia, por lo que es reconocida como la primera programadora.

* ### La Máquina De Turing ###
Definida por el matemático inglés Alan Turing como una "máquina automática" en 1936.  
Modelo teórico que usa una cinta infinita y reglas para procesar información. Demuestra que cualquier cálculo puede realizarse con las instrucciones adecuadas, dando origen a la computación universal donde lo importante es el código.

## DEMOCRATIZACIÓN DEL CREATIVE CODING ##

* ### Software lIBRE ###
Surge en 1983 con el inicio del proyecto GNU (Richard Stallman), promoviendo sistema operativo libre. En 1985 se publica el Manifiesto GNU y se funda la Free Software Foundation (FSF). Defiende 4 libertades:  
1. Libertad 0 (Uso)
2. Libertad 1 (Estudio)
3. Libertad 2 (Distribución)
4. Libertad 3 (Mejora)
Además de principios adicionales como Copyleft, Libertad no precio, Lucha contra restricciones y desarrollo del proyecto GNU.

| Free Software (Software Libre) | Open Source (Código Abierto)|
| :-- | --: |
| Ético y Social. Se trata de derechos y libertades morales | Práctico y Técnico. Se trata de una metodología de desarrollo eficiente |
| La libertad del usuario es lo más importante | La calidad del software y la colaboración son lo más importante |
| "Libertad" El usuario debe tener el control de su informática | "Colaboración" Ek código abierto permite que más ojos revisen errores y mejoren el producto |
| El software privativo (propietario) se considera un prioblema ético o antisocial | El software privativo es aceptable, pero el modelo abierto se considera técnicamente superior |
| FSF (Free Software Foundation) | OSI (Open Source Initiative) |
| Richard Stallman (Fundador de GNU) | Eric S. Raymond / Bruce Perens |

* ### Design By Numbers ###
En 1999, John Maeda creó un lenguaje en el MIT Media Lab para enseñar a diseñadores visuales a programar.  
"El problema de usar herramientas como Photoshop es que estás limitado a lo que los ingenieros de Adobe imaginaron que podrías hacer. Cuando aprendes a programar, tú creas tus propias herramientas"

* ### Processing ###
Es un software libre de código abierto y gratuito creado en 2001 por Ben Fry y Casey Reas en el MIT Media Lab. Permite crear gráficos, animaciones e interacciones, ampliando Design by Numbers agregando paleta de color, tamaño variable y manteniendo la simplicidad en la sintaxis en el código y la idea del "Sketchbook"
